This plugin allows you to sign into the WordPress admin portal using Microsoft Entra ID SSO.

== Description ==
This plugin can be used to sign in to WordPress using Microsoft credentials.

== Installation ==
1. Go to the Azure portal.
2. Create an app registration.
3. Setup the app registration (audience, redirect uri).
4. Copy the values from the app registration on the plugin option name.
5. Store the values in their respective fields.
